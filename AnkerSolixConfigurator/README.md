# Anker Solix Konfigurator

Erkennt automatisch alle Geräte der verbundenen Anker Solix Anlage und ermöglicht das Anlegen von Device-Instanzen per Klick.

## 1. Voraussetzungen

- IP-Symcon ab Version 8.1
- Eingerichtetes **Anker Solix IO**-Modul mit gültigen Zugangsdaten

## 2. Einrichtung

1. **Anker Solix Konfigurator** anlegen  
   → IP-Symcon fragt automatisch, ob ein vorhandenes IO-Modul ausgewählt oder ein neues angelegt werden soll
2. Im IO-Modul Zugangsdaten eintragen und Verbindung testen
3. Konfigurator öffnen — alle Geräte der Anlage erscheinen in der Liste
4. Gewünschtes Gerät markieren → **Erstellen**  
   → Die Device-Instanz wird vollständig vorbefüllt angelegt und startet sofort

## 3. Gerätetypen

Der Konfigurator erkennt folgende Gerätetypen automatisch:

| Typ | Beschreibung |
|---|---|
| **Solarbank** | SOLIX E1600, E1600 Pro u. a. |
| **Smart Plug** | SOLIX Smart Plug |
| **Smart Meter** | Energiemessgerät am Netzanschluss |
| **Powerstation** | SOLIX C1000, C800 u. a. |
| **Home Power** | SOLIX X1 und weitere Heimspeicher |
